§5 PHƯƠNG TRÌNH BẬC NHẤT ĐỐI VỚI $\sin x$ VÀ $\cos x$

A. LÝ THUYẾT

Dạng $a \sin x + b \cos x = c$ ($a^2 + b^2 \neq 0$)

Cách giải

Chia hai vế của phương trình cho $\sqrt{a^2 + b^2}$, phương trình trở thành:

$$ \frac{a}{\sqrt{a^2 + b^2}}\sin x + \frac{b}{\sqrt{a^2 + b^2}}\cos x = \frac{c}{\sqrt{a^2 + b^2}} $$

Vì $\left(\frac{a}{\sqrt{a^2 + b^2}}\right)^2 + \left(\frac{b}{\sqrt{a^2 + b^2}}\right)^2 = 1$ nên có góc $\alpha$ sao cho:

$$ \cos\alpha = \frac{a}{\sqrt{a^2 + b^2}} \text{ và } \sin\alpha = \frac{b}{\sqrt{a^2 + b^2}} $$

Ta có phương trình tương đương: $\sin x \cos \alpha + \cos x \sin \alpha = \frac{c}{\sqrt{a^2 + b^2}}$

Áp dụng công thức cộng, ta được phương trình:

$$ \sin(x + \alpha) = \frac{c}{\sqrt{a^2 + b^2}} $$

Dễ dàng giải được phương trình này.

Nhận xét

  1. Phương trình $a \sin x + b \cos x = c$ có nghiệm khi và chỉ khi $a^2 + b^2 \ge c^2$.
  2. Các phương trình $a \sin x - b \cos x = c$, $a \cos x \pm b \sin x = c$ cũng được giải tương tự.

B. BÀI TẬP

1.1. Giải các phương trình sau:

  1. $\sqrt{3}\sin x - \cos x = 1$
  2. $\sqrt{3}\cos 3x - \sin 3x = 2$
  3. $3\cos x + 4\sin x = -5$
  4. $\sin x - 7\cos x = 7$
  5. $2\sin 2x - 2\cos 2x = \sqrt{2}$
  6. $\sin 2x = \sqrt{3} - \sqrt{3}\cos 2x$

1.2. Giải các phương trình sau:

  1. $2\sin^2 x + \sqrt{3}\sin 2x = 3$
  2. $2\cos^2 x - \sqrt{3}\sin 2x = \sqrt{2}$
  3. $2\sin 2x \cos 2x + \sqrt{3}\cos 4x + \sqrt{2} = 0$
  4. $4\sin^2 x + 3\sqrt{3}\sin 2x - 2\cos^2 x = 4$

1.3. Giải các phương trình sau:

  1. $\sin 3x - \sqrt{3}\cos 3x = 2\cos 4x$
  2. $\cos x - \sqrt{3}\sin x = 2\cos(\frac{\pi}{3} - x)$
  3. $\sqrt{3}\sin 2x + \cos 2x = \sqrt{2}\cos x - \sqrt{2}\sin x$
  4. $\sin 8x - \cos 6x = \sqrt{3}(\sin 6x + \cos 8x)$

1.4. Giải các phương trình sau:

  1. $3\sin(x - \frac{\pi}{3}) + 4\sin(x + \frac{\pi}{6}) + 5\sin(5x + \frac{\pi}{6}) = 0$
  2. $\sqrt{2}\sin(x + \frac{\pi}{4}) + 4\sin(x - \frac{\pi}{4}) = \frac{3\sqrt{5}}{2}$

1.5. Giải các phương trình sau:

  1. $3\sin x - \sqrt{3}\cos 3x = 1 + 4\sin^3 x$
  2. $\sqrt{3}\cos 5x - 2\sin 3x \cos 2x - \sin x = 0$
  3. $(\sin\frac{x}{2} + \cos\frac{x}{2})^2 + \sqrt{3}\cos x = 2$
  4. $8\cos 2x = \frac{\sqrt{3}}{\sin x} + \frac{1}{\cos x}$

1.6.

Tìm $x \in (\frac{2\pi}{5}, \frac{6\pi}{7})$ thỏa phương trình $\cos 7x - \sqrt{3}\sin 7x = -\sqrt{2}$.

1.7.

Cho phương trình $2\sin^2 x - \sin x \cos x - \cos^2 x = m$.

  1. Tìm m để phương trình có nghiệm.
  2. Giải phương trình với $m = -1$.

1.8.

Cho phương trình $\sin 2x - 2m\cos x = \sin x - m$. Tìm m để phương trình có đúng hai nghiệm thuộc đoạn $[0; \frac{3\pi}{4}]$.

1.9. Giải các phương trình sau:

  1. $8\sin x = \frac{\sqrt{3}}{\cos x} + \frac{1}{\sin x}$
  2. $2\sqrt{\sin x} = \frac{\sqrt{3}\tan x}{2\sqrt{\sin x} - 1} - 1$
